# Sales-Analytics
Sales analytics project focuses on how customers interact with retails product across differents locations. By analyzing retail transactional data, the project identifies key customer orders, sales performance, and product affinities. these insights help businesses make informed decisions about marketing and customer retention strategies.

## Project Objectives
The project aims to:
- Analyze sales distrubtion by category, sub-category amd product performance
- Study customer purchase behavior base on quantity order, state amd city
- Evaluate monthly/annuanly sales and profit trends and identify seasonal appaterns
- compare high profit vs low profit category
- Identify the most preferred payment methods.

## Data Preparation & Cleaning
- Handle missing value, inconsistencies and data types
- Convert dates into proper format and extract and time feature ( month, year, weeks and quarters)
- create new calculated columns (first purchase date and profit margin)
